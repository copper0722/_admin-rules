## Hard rules (from wiki/CLAUDE.md)

- **raw.md = machine-complete**: every section, table, image description. CC reads raw.md instead of PDF.
- **Wiki = M2M compressed .md**: zh-TW Taiwan academic terminology, never PRC.
- **File naming**: `wiki_{topic}_{subtopic}.md` lowercase underscore-separated.
- **Frontmatter required**: `type`, `scope`, `source_docs`, `zotero_keys`, `status`, `generated`.
- **Causal/EBM mindset**: flag confounding/competing risk/immortal time per Hernán + Guyatt checklist.
- **>500L file**: add `## KEY TAKEAWAYS` first or `summary:` frontmatter.
- **>2000L file**: split into logical parts.
- **No `[[wikilink]]`** in new files (vault uses standard markdown `[text](/abs/path)`).
- **No "Generated by AI" disclaimer** in wiki content (Copper preference).
