# FB Pipeline - Public Policy

This public card covers content drafting only. Publishing credentials, page tokens, local token paths, private draft folders, and automation details must stay private.

## Drafting Rules

- Do not impersonate a human author.
- Include AI-authorship disclosure when the post is AI-generated or AI-edited.
- Cite public sources and avoid paywalled quotations.
- Do not publish patient data, private messages, or copyrighted source text.
- Keep final publication human-reviewed unless the private runtime explicitly authorizes automation.

## Language Style (Copper directive 2026-05-11)

FB posts target zh-TW readers (TLC Medicine / 一般民眾 + 醫療專業混合
audience). Apply these style rules to every draft delivered for review:

- **No em-dash connectors** (`——`, `—`, `--`) used as sentence joiners.
  Copper reads em-dash usage as an AI tell and dislikes it. Replace with
  conventional zh-TW punctuation: `，`, `：`, `；`, `。`, or paragraph
  break. A single `—` alone on its own line as a visual horizontal rule
  is acceptable; an em-dash inside a sentence to introduce an aside is
  not.
- **zh-TW default, restrained inline English.** Translate any concept
  that has a natural zh-TW equivalent — including:
  - `nephrologist` → 腎臟科醫師
  - `race correction / race-adjusted / race-corrected` → 種族校正
  - `audit` (verb) → 重新檢視 / 稽查
  - `lecture` → 課堂
  - `troublemaker` → 麻煩人物
  - `epiphany` → 頓悟
  - `social media` → 社群媒體
  - `health disparities researcher` → 健康不平等研究者
  - `policy` → 政策
  - `class 2015` (medical school) → 醫學院 2015 年入學
  - `chief` (chief resident / 內科主任) → 總醫師 / 主任 per context
- **Inline English kept** for: scientific abbreviations / units / formulas
  (eGFR, MDRD, CKD, KDIGO, CKD-EPI), proper nouns (Harvard, NEJM, BIDMC,
  UW, Tufts, COVID, George Floyd, Instagram), people names, podcast /
  series / organization titles (Intention to Treat, The Race Equation,
  White Coats for Black Lives, Medical MythBuster), degree suffixes (MD,
  PhD), and DOI strings.
- **First-mention parenthetical translation** acceptable when a
  jargon-loaded English term has no widely-known zh-TW gloss: `race
  correction（種族校正項）` first time, then 種族校正 alone afterwards.
- Avoid mixed-language compound phrases when a pure-zh phrasing exists
  (`race-adjusted 公式` → `種族校正公式`).
- Quote translation: when quoting an interviewee, translate to zh-TW
  rather than quoting English verbatim, unless the English itself is the
  rhetorical point (rare).

## Public Output

Drafts may contain:

- Post title or hook.
- Plain-language explanation.
- AI-authorship disclosure.
- Suggested image/card concept using original wording.
- Source citation by name + date (NOT a URL).

Drafts must not contain:

- `http://` or `https://` URLs in the post body. Universal rule per Copper
  2026-05-14: Facebook algo systematically penalises posts with external
  links, organic reach collapses. Applies to 社團 / 粉專 / personal wall
  alike. To direct readers: cite source by name + date, place URL in the
  first comment after publishing (comments are not penalised), or point
  readers to a non-FB platform link tree. The only inline links permitted
  are FB-internal kinds (tagged users, location check-ins).
- Access tokens, private paths, unpublished personal records.
